NicorepoChecker
===============
ニコレポチェッカーはChromeの拡張機能として機能し、  
ニコレポの新着情報をポップアップ表示します。


開発環境の設定
--------------
Chromeメニュー→ツール→拡張機能から  
[パッケージ化されていない拡張機能を読み込む...]ボタンで  
"nicorepo"フォルダを指定して[OK]


crx(配布)ファイルの作成
-----------------------
Chromeメニュー→ツール→拡張機能から  
[拡張機能のパッケージ化...]ボタンで  
拡張機能のルートディレクトリに"nicorepo"フォルダを指定  
[拡張機能のパッケージ化]ボタンを押すとcrxファイルとpemファイルが作成されます。  


crxファイルからのインストール
-----------------------
Chromeメニュー→ツール→拡張機能を開き  
配布されているcrxファイルをドラッグアンドドロップして  
拡張機能窓に放り込むだけです。


使い方(設定)
------------
Chromeメニュー→ツール→拡張機能を開き  
ニコレポチェッカーのオプションリンクを開きます。  
ニコニコにログインしていれば自動でお気に入りユーザーとコミュニティを取得します。

"すべて"のチェックボックスをON/OFFすることで全員のチェックを付けたり外したり出来ます。  
"新規"のチェックボックスをONにしておくことで新たに増えたメンバーにその設定をコピーします。


棒読みちゃんとの連携
--------------------
websocketを使用して棒読みちゃんに話させることができるのですが、  
棒読みちゃん用websocketプラグインを公開していないのでまだ連携出来ません。  
整理でき次第公開するかもしれません。
